# Feature Implementation Guide

For every feature follow this exact order.

1. Read Product Documentation.

2. Read Database Documentation.

3. Read UI Documentation.

4. Create Folder Structure.

5. Create Entity.

6. Create Model.

7. Create Repository Interface.

8. Create Repository Implementation.

9. Create Providers.

10. Create Reusable Widgets.

11. Build Screen.

12. Connect Navigation.

13. Connect Supabase.

14. Handle Errors.

15. Handle Loading.

16. Handle Empty State.

17. Add Validation.

18. Test Feature.

19. Update Documentation.

20. Commit.
