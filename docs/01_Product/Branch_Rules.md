# Branch Rules

## Branch Ownership

Every repair order belongs to one branch.

---

## Operator

Each branch has one operator account.

Only that operator manages daily work.

---

## Technicians

Each branch manages its own technicians.

Technicians cannot belong to another branch.

---

## Isolation

Branch operators cannot access:

Other branches.

Other customers.

Other repair orders.

Other reports.

---

## Administrator

Administrator can access all branches.
