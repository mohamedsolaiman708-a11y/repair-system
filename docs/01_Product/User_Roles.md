# User Roles

## Administrator

The Administrator owns the entire system.

Permissions include:

Manage branches.

Manage branch operators.

Manage system settings.

Manage WhatsApp templates.

View all reports.

View all repair orders.

View all customers.

Manage subscription (future).

The Administrator can access all branches.

---

## Branch Operator

Each branch has only one operator.

The operator performs all daily operations.

Responsibilities:

Register customers.

Register devices.

Assign technicians.

Update repair status.

Receive customer payments.

Deliver repaired devices.

Search repair orders.

Print receipts.

Generate barcodes.

The operator cannot access other branches.

---

## Technician

Technicians DO NOT access the system.

The assigned technician is stored only as information inside the repair order.

The operator updates repair progress on behalf of technicians.

---

## Customer

Customers do not access the system.

Communication occurs through WhatsApp notifications and printed receipts only.
