# WhatsApp Events

## Purpose

Defines every event that triggers an automatic WhatsApp message.

---

## Event

Repair Registered

Trigger

Repair order created.

Message

Device received confirmation.

---

## Event

Inspection Finished

Trigger

Inspection completed.

Message

Inspection results.

---

## Event

Waiting Customer Approval

Trigger

Quotation entered.

Message

Customer approval request.

---

## Event

Repair Completed

Trigger

Status becomes Ready For Pickup.

Message

Device ready for collection.

---

## Event

Device Delivered

Trigger

Status becomes Delivered.

Message

Thank you message.

---

## Event

Pickup Reminder

Trigger

Device stays Ready For Pickup for configured days.

Message

Reminder.
