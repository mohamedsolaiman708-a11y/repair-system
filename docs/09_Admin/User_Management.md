# User Management

## Purpose

Manage administrator and branch operator accounts.

---

## Features

Create User

Edit User

Deactivate User

Reset Password

Assign Branch

Assign Role

---

## Roles

Administrator

Branch Operator

---

## User Information

Full Name

Email

Phone

Role

Branch

Status

Last Login

Created Date

---

## Rules

Each branch can have only one active operator.

Administrators have access to all branches.

Operators can only access their assigned branch.
